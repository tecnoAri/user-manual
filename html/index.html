Manual de Usuario

¡Bienvenido al manual de usuario de nuestra aplicación! Aquí encontrarás toda la información que necesitas para comenzar a usar nuestra aplicación de manera efectiva.

Inicio de Sesión

El inicio de sesión es el proceso mediante el cual puedes acceder a tu cuenta en nuestra aplicación. Para iniciar sesión, sigue estos pasos:

  1. Acceso a la Página de Inicio de Sesión

Para comenzar, ve a la página de inicio de sesión. Desde allí, los usuarios pueden iniciar sesión en cuentas existentes o crear nuevas.

  1. Acceso a la Interfaz de Creación de Cuenta:

Cuando llegues a la página de inicio de sesión, encontrarás una interfaz para crear nuevas cuentas. Aquí comienza el proceso de registro.

Una vez que hayas creado tu cuenta en nuestra aplicación web, recibirás un correo electrónico con un código de verificación. Después de ingresar este código de verificación en la aplicación, tu cuenta se activará y serás redirigido automáticamente a la página de inicio de la aplicación web.

Si en algún momento deseas cerrar sesión, puedes hacerlo con facilidad. Luego, si decides iniciar sesión nuevamente, simplemente utiliza la cuenta que has creado previamente para acceder a todas las funcionalidades de nuestra aplicación.
